🚲

Pipeline Data Engineering — Vélib (Open Data Paris)

Ingestion API • Transformation • BigQuery • Airflow • Docker

Conception d’un pipeline de data engineering de bout en bout permettant d’ingérer les données temps réel des stations Vélib via une API open data, de les transformer puis de les stocker dans une architecture cloud analytique. Le pipeline est automatisé et conçu pour être robuste (idempotence) et optimisé pour la requête.

1️⃣ Ingestion
API REST (vélos, docks, timestamp) • Python
2️⃣ Transformation
Normalisation JSON → schéma tabulaire • typage • ingestion_time • nettoyage
3️⃣ Stockage
BigQuery : Staging (WRITE_TRUNCATE) + Final (MERGE idempotent, clé logique)
4️⃣ Orchestration
Airflow : DAG planifié • exécutions batch • automatisation
Python BigQuery Airflow Docker SQL MERGE Partitionnement
